- 这个每个人的项目不同，覆盖的技术也不一样，所以不能统一去说。- 无非是找到自己项目中的亮点，简历上叙述的简练并且吸引眼球，同时自己要很熟悉这个点(毕竟可以提前准备)- 最好自己多练，就像有个剧本或者稿子一样，保证面试中可以很熟练通俗地讲出，并且让人听着很舒服。
